# CLAUDE.md

Ты работаешь над сайтом «Школа ИКС — Центр знаний».

Всегда перед изменениями читай:
- docs/00_PROJECT_CONTEXT.md
- docs/01_PRODUCT_ARCHITECTURE.md
- docs/02_SITE_STRUCTURE.md
- docs/03_DESIGN_SYSTEM.md
- docs/05_FEATURES.md
- docs/06_BILLING.md (источник истины по списаниям)

Главная задача:
генерировать и улучшать сайт на основе всех прошлых, текущих и новых данных проекта.

Стиль:
- тёмная neon purple стилистика
- современный образовательный лендинг
- доверие родителей + интерес детей
- чистая структура, без перегруза
- адаптивность: desktop / tablet / mobile

Нельзя:
- ломать существующую структуру без причины
- удалять данные из docs без переноса в changelog
- менять визуальный стиль без обновления DESIGN_SYSTEM
- придумывать факты о школе, которых нет в docs

После каждого крупного изменения:
1. обнови docs/09_CHANGELOG.md
2. проверь адаптивность
3. проверь CTA и формы
4. кратко объясни, что изменено
