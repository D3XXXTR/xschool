# 03 — Дизайн-система

Источник истины — `src/assets/css/tokens.css` и локальный `<style>` в `src/index.html`.
Этот документ — обзор и руководство к использованию токенов.

## 1. Общая идея

Тёмная база с фиолетовым neon-акцентом. **Не киберпанк** — современный
образовательный SaaS: доверительный, мягкий, premium. На публичной
странице упор на ребёнка-доверие, а не на интерфейсные карточки.

## 2. Фон и поверхности

| Назначение | Значение |
|---|---|
| Основной фон публички | `#0c0a1f` (см. локальный `:root` в `index.html`) |
| Фон кабинетов | `--bg: #07061a` (`tokens.css`) |
| Поверхности секций | `rgba(255,255,255,.04)` |
| Карточки активные | `rgba(255,255,255,.05)` |
| Карточки на hover | `rgba(255,255,255,.07)` |
| Бордер по умолчанию | `rgba(255,255,255,.08)` |
| Бордер на hover | `rgba(255,255,255,.16)` или акцентный |

**Halo-свечения** в hero — два радиальных градиента (`radial-gradient`)
с opacity 0.16-0.22. Без `filter: blur()` — это лагает на mobile.

## 3. Цвета направлений

| Направление | Slug | Линия / акцент |
|---|---|---|
| Школа | `school` | `linear-gradient(90deg, #a78bfa, #ec4899)` |
| IT-Академия | `academy` | `linear-gradient(90deg, #5eead4, #7dd3fc)` |
| ОГЭ / ЕГЭ | `exam` | `linear-gradient(90deg, #fcd34d, #f97316)` |

Цвета направлений на публичной странице — **просто визуальные акценты карточек**, не привязаны к `body[data-mode]`. На лендинге `data-mode` не используется.

В кабинетах сохранены `body[data-mode="academy"]` и `body[data-mode="exam"]` (см. `tokens.css`) — но это опциональная функция, в текущем MVP не подмешивается.

## 4. Цвета статусов

| Роль | Hex | Где используется |
|---|---|---|
| Success | `#10b981` / `#86efac` | прогресс ≥85%, статус «оплачено», бейдж «пробное бесплатно» |
| Warning | `#fbbf24` / `#fcd34d` | прогресс 60-85%, ожидающие платежи, низкий остаток |
| Danger | `#ef4444` | просрочки, пакет = 0, удаления |
| Info | `#3b82f6` / `#7dd3fc` | бейдж «Завтра», нейтральные информационные плашки |

## 5. Типографика

| Шрифт | Где |
|---|---|
| Unbounded (400-900) | заголовки, кнопки, метрики, акценты |
| Nunito (300-800) | весь body-текст, формы, подписи |

Размеры (адаптивные):
| Элемент | Размер |
|---|---|
| Hero H1 | `clamp(36px, 5vw, 64px)`, weight 800 |
| Section H2 | `clamp(32px, 4.6vw, 48px)`, weight 800 |
| Card H3 | 22-24 px, weight 700-800 |
| Body | 14-16 px, weight 400-500 |
| Caption / muted | 12-13 px, weight 500 |
| Trust / badge | 11-13 px, weight 600-700, letter-spacing 0.04-0.18em |

## 6. Кнопки

```
.btn-p   — primary, gradient #7c3aed → #ec4899, white text
.btn-s   — secondary, transparent, border 1px solid rgba(255,255,255,.08)
.dir-cta — кнопка направления, цвет под слот (school/academy/exam)
.btn-nav — кнопка в шапке (Войти), gradient
.nav-teacher — outline, муть-цвет
```

| Свойство | Значение |
|---|---|
| Радиус | 10-12 px (стандартные кнопки), 14-16 px (CTA в hero и блоках) |
| Padding | 10-26 px по высоте, 14-24 px по ширине |
| Hover | `transform: translateY(-1px)` + усиление shadow + brightness(1.08) |
| Active | `transform: translateY(0)` + scale(0.98) |
| Transition | 200 ms ease-out |

## 7. Карточки

**Карточка направления (#directions)**: 22 px радиус, верхняя цветная полоска
4 px (градиент по слоту), pill-тег направления, H3, описание, 3 буллета,
CTA полная ширина, под кнопкой подпись «✓ Пробное бесплатно».

**Карточка эксперта (#experts)**: 20 px радиус, верхняя полоска 4 px, фото-плейсхолдер 1:1 с инициалами на градиенте, имя, предметы фиолетовым,
бейдж опыта (pill), краткое достижение, теги. **Без рейтинга, без цены, без CTA внутри карточки** — действия идут только через формы заявки.

**Карточка отзыва (#reviews)**: 20 px радиус, иконка-цитата `❝` в правом верхнем углу, 5 жёлтых звёзд, текст, аватар-инициалы (цветной круг 42 px),
имя + роль/контекст.

**Карточка кабинетов** (`.row-card`, `.package-card` и т.п. в `main.css`):
14 px радиус, более плотный padding, для информационных списков и таблиц.

## 8. Hover и переходы

Везде один таймер: **200 ms ease-out** (180-250 ms допустимо).

| Элемент | Эффект |
|---|---|
| Кнопки | `translateY(-1px)` + brightness(1.08) + box-shadow усиление |
| Карточки | `translateY(-2px)` + светлеет фон + border-color → акцент |
| Pill / tab | смена `background` и `color`, без transform |
| Ссылки в шапке | `color → var(--text)` |

Запрещены тяжёлые анимации: blur, scale > 1.05, transitions длиннее 400 ms.

## 9. Адаптив

Брейкпойнты (используются в `main.css` и `index.html`):

| Размер | Поведение |
|---|---|
| ≥1280 px | Полная ширина: sidebar в кабинетах, 4 в ряд в экспертах, 3 в ряд в направлениях/отзывах |
| 1024-1280 | sidebar остаётся, эксперты 2 в ряд (узкий desktop) |
| ≤1024 | sidebar скрыт, появляется bottom-nav (5 пунктов), отзывы и пакеты в одну колонку, эксперты 2 в ряд |
| ≤960 | направления в 1 колонку, локация в 1 колонку |
| ≤640 | header 56 px, эксперты в 1 колонку, footer в 1 колонку, модалки полная ширина |

## 10. Иконки

Минимум emoji в коде (используются точечно: ✓ для бейджа пробного,
❝ для цитаты в отзывах). Базовые иконки — inline SVG (логотип ИКС в шапке, пин в карте).

## 11. Удалено из дизайн-системы

- **Mode-switcher** на публичной странице — не используется. Атрибут `body[data-mode]` остаётся в `tokens.css` для будущего опционального применения в кабинетах.
- **Уведомления (Notifications)** — модуль удалён, нет `notify.js`, нет inbox, нет бейджей непрочитанного. Сигналы передаются через UI-баннеры, бейджи статусов и истории.
- **Glassmorphism** упрощён: остался только в светлых floating-виджетах поверх halo (если возвращать hero-композицию).
